A simple expense tracking web application built using ASP.NET Core MVC, Entity Framework Core, and Chart.js.

## Features
- Add and manage expenses
- Categorize expenses
- View expenses in a table
- Visualize spending with charts (Chart.js)
- EF Core database integration

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server 
- Chart.js

## Purpose
Built as a learning project to practice full-stack ASP.NET Core development and MVC architecture.
